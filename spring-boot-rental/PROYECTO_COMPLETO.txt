🚀 CÓMO EMPEZAR
═════════════════════════════════════════════════════════════════════════════

OPCIÓN 1: Desarrollo Local (3 pasos)
────────────────────────────────────
1. Crear BD: mysql -u root -p < database/schema.sql
2. Instalar: mvn clean install
3. Ejecutar:  mvn spring-boot:run
4. Abrir:     http://localhost:8080

OPCIÓN 2: Despliegue en Railway
──────────────────────────────
1. Crear repo en GitHub
2. Crear proyecto en Railway.app
3. Conectar repositorio
4. Agregar MySQL
5. Ver DEPLOYMENT.md para detalles
